CORS(Cross-Origin Resource Sharing)
: 한 origin에서 실행중인 웹 어플리케이션이 다른 출처의 리소스에 접근 할 수 있도록 허용하는 메커니즘

CORS는 클라이언트 측 보안정책이다.
주로 브라우저가 JS를 통해 외부 서버에 API를 요청할 때 적용

하지만,서버에서 다른 서버로의 API 요청은 CORS 정책의 적용을 받지 않는다.

즉,서버 대 서버 통신에서는 CORS가 브라우저처럼 중재하지 않기 때문에 제한없이 자유롭게 통신 할 수 있다.



WebClient
Spring5에서 도입된 비동기 HTTP 클라이언트로,RESTful 웹서비스와 통신을 비동기적으로 처리할 수 있는 강력한 도구이다.

HTTP 요청 메서드를 지원
GET,POST,PUT,DELETE를 다 지원한다.

WebClient의 반환타입으로 Mono타입,Flux타입이 있다.

Mono: 0 또는 1개의 데이터를 반환
Flux: 0개 이상의 데이터를 반환
