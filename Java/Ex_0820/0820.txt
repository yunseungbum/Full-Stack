File클래스
isFile() -> 최종목적지가 파일이면 true
isDirectory() -> 최종목적지가 폴더이면 true
exist() -> 최종목적지가 존재하면 true
length() -> 최종목적지의 크기를 반환
list() -> 최종목적지 아래의 목록을 String배열로 반환
mkdirs() -> 폴더를 만들어줌

문자변환 보조스트림
-바이트 기반 스트림으로 텍스트 파일을 읽거나 쓸 경우, 한글을 포함한
 비영어권 문자들이 정상적으로 출력이 되지 않는 모습을 보았다.
-소스 스트림이 바이트 기반이고, 입출력 데이터가 문자라면 Reader로
 변환하여 사용하는 것을 고려해 봐야한다.

InputStreamReader
-InputStreamReader는 바이트 기반 스트림 
 InputStream을 문자 기반의 스트림 Reader로 변환하는 보조 스트림이다.


열거형
-상수를 가지고 생성되는 객체들을 한곳에 모아둔 하나의 묶음
-상수를 사용자가 지정한 이름으로부터 0부터 순차적으로 증가(index)
-클래스처럼 보이게 하는 상수
-관련있는 상수들을 모아서 배열처럼 정의



enum은 데이터의 그룹화 및 관리에 용이
-데이터들이 서로 관려되어 있지만 관련된 형태를 구현하는데 있어 에러사항이 생긴다면
 enum을 통해 한 클래스 내에서 관리할 수 있다.
